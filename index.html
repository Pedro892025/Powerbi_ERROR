Aviso de Perda de Dados - Power BI

Atenção: Todas as bases de dados utilizadas no Power BI foram perdidas permanentemente. Isso inclui os conjuntos de dados destinados ao CPV e também os utilizados para as reuniões de preços.

O sistema realizou uma exclusão automática de todos os arquivos, e também foram apagadas todas as cópias de segurança, tanto as armazenadas em locais remotos quanto as salvas diretamente na área de trabalho.

Você precisará refazer toda a estrutura e os dados do zero, pois não há nenhuma forma de recuperação.
